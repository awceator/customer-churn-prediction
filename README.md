# 📊 Customer Churn Analysis Dashboard

A complete end-to-end Data Analytics project that analyzes customer churn in the telecom industry using **Google Colab, Python, Google Sheets, and Looker Studio**. This project helps identify customer behavior, churn patterns, and key business insights through interactive dashboards.

---

## 📌 Project Overview

Customer churn is one of the biggest challenges for telecom companies. This project analyzes customer data to identify the factors that influence churn and provides an interactive dashboard for business decision-making.

---

## 🎯 Objectives

- Analyze customer churn patterns
- Identify high-risk customer segments
- Visualize customer demographics
- Analyze contract types and payment methods
- Monitor customer retention
- Support data-driven business decisions

## 📊 Business Insights

- Month-to-month contract customers have the highest churn rate.
- Fiber optic users churn more frequently than DSL users.
- Customers with shorter tenure are more likely to churn.
- Electronic Check is the most common payment method among churned customers.
- Higher monthly charges are associated with increased churn.
